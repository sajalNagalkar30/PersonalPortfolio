# Sajal Nagalkar — Personal Portfolio

A personal portfolio website built with React showcasing my skills, experience, projects, and achievements.

## Live Demo

> https://sajalNagalkar30.github.io/PersonalPortfolio

## Features

- Responsive design for all screen sizes
- Smooth scroll animations
- Typing effect in hero section
- Sections: Hero, About, Skills, Experience, Projects, Achievements, Services, Education, Contact
- Scroll to top button
- Clean and minimal UI

## Tech Stack

- **React 19**
- **React Icons**
- **CSS3** (custom stylesheets per section)
- **Create React App**

## Getting Started

### Prerequisites

- Node.js >= 14
- npm >= 6

### Installation

```bash
# Clone the repository
git clone https://github.com/sajalNagalkar30/PersonalPortfolio.git

# Navigate into the project
cd PersonalPortfolio/sajal-portfolio

# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```
